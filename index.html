How this works: this page calls the free, public Open-Meteo forecast API directly from your browser — no API key, no server in between. For each of the next 7 days it looks only at the 8am–7pm window, scoring gusty wind, rain hours, and too-hot (>27°C) or too-cold (<10°C) temperatures. It's a live forecast, not a saved dataset, so later days in the week can still shift as the date gets closer — same as any weather app. Forecast source: open-meteo.com.
